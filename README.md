# Student Management System

A Python application for managing student records in a PostgreSQL database with full CRUD operations.

## Features

- View all students
- Add new students
- Update student emails
- Delete students
- Database setup and initialization

## Files

- database directory contains sql files to set up database
- main.py is a simple script to test each function
- operations.py contains all the functions for the CRUD operations, database connection, and query execution

## Requirements

- Python 3.8+
- PostgreSQL 12+
- psycopg2 (On Ubuntu, use 'sudo apt install python3-psycopg2' to install)
